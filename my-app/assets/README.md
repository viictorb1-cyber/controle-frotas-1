# Assets

Coloque os seguintes arquivos nesta pasta:

- `icon.png` - Ícone do app (1024x1024 px)
- `splash-icon.png` - Ícone da splash screen (512x512 px)
- `adaptive-icon.png` - Ícone adaptativo Android (1024x1024 px)
- `favicon.png` - Favicon para web (48x48 px)

Você pode gerar esses assets usando o Figma ou qualquer editor de imagens.

Para um teste rápido, você pode criar imagens placeholder coloridas.
